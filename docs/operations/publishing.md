# コンテンツの編集・公開

## 現状

既存のWordPressサイトは自由に編集しながら作成してきたため、統一された編集ルールがない。

## 今後の方針

- コンテンツはMarkdownで管理する。
- 編集ルールを別途言語化し、仕様としてまとめる。
- カテゴリごとの重要情報を記事上部へ配置する。
- 同じカテゴリでも重要項目の順序は記事ごとに異なるため、過度に固定的なルールにはしない。
- 既存記事はリニューアル版の公開後に順次リライトする。
